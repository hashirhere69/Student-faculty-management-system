# Student-faculty-management-system
A web-based dashboard system that allows students and faculty to register, manage profiles, and store academic information using a backend framework with MySQL/PostgreSQL database integration.

#Objectives
The main objectives of this project are:

• Provide a centralized dashboard for student information
• Allow faculty members to store and manage professional records
• Integrate backend services with a relational database
• Demonstrate full-stack web development with database interaction

#Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python Flask / Django / Node.js / Spring Boot (whichever you used)

Database:
- MySQL / PostgreSQL

#Database Design
Students Table
- student_id
- name
- email
- password
- academic_information

Faculty Table
- employee_id
- name
- email
- department

2. Navigate to project directory

cd student-faculty-management-system
